// Purpose statement for the assembly
As a business analyst or business rules developer, you can design custom {CENTRAL} pages.

.Prerequisites
* You are logged in to {CENTRAL} as a user with permission to edit pages.

== Custom {CENTRAL} pages

You can use the *Page Editor* in {CENTRAL} to create static pages or pages that contain dynamic reporting components. Dynamic pages are called dashboard pages. You can customize the component properties of a page according to your requirements. Optionally, before you create a dashboard page, you can define a data set to feed the reporting components. You can export dashboard pages to the standalone Dashbuilder Runtime viewer for on premise installations or the Dashbuilder Standalone viewer on {OPENSHIFT}.
