In {PRODUCT}, you can view all process instances from the *Manage Process Instances* page. As an admin, you can filter through process instances in {CENTRAL}.

.Prerequisites
* Installed Red Hat JBoss Enterprise Application Platform 7.1.0. For more information, see {URL_INSTALLING_ON_EAP}[_{INSTALLING_ON_EAP}_].
* Installed {PRODUCT}. For more information, see the {URL_INSTALLING_DM_ON_PREMISE}[_{installing_red_hat_decision_manager_on_premise}_].
* {PRODUCT} is running and you can log in to {CENTRAL} with the `admin` role.
