// Purpose statement for the assembly
As an admin, you can customize {CENTRAL} using the `Admin Settings`. In *Settings*, you can modify the following aspects of {CENTRAL} to customize it as per your requirements:

* `Roles`: View and set home page, priority, and permissions of a role.
* `Groups`: View and set home page, priority, and permissions of a group. You can also view, create, and delete groups using this option.
* `Users`: View, create, and delete users using this option. You can add a group, role and view the permissions of a user.
* `Artifacts`: View M2 repository artifacts, upload artifacts, view, and download *.jar files.
* `Data Source`: Add, update, or delete data sources and database drivers.
* `Data Sets`: Create, modify or delete data sets.
* `Projects`: View and edit project preferences, such as, file export properties, space properties, default values, and advanced GAV properties.
* `Artifact Repository`: Manage artifact repository properties using this option.
* `Languages`: Set {CENTRAL}'s language.
* `Process Designer`: Set diagram editor properties using this option.

.Prerequisites
* Installed Red Hat JBoss Enterprise Application Platform 7.1.0. For more information, see {URL_INSTALLING_ON_EAP}[_{INSTALLING_ON_EAP}_].
* Installed {PRODUCT}. For more information, see the {URL_INSTALLING_DM_ON_PREMISE}[_{INSTALLING_DM_ON_PREMISE}_].
* {PRODUCT} is running and you can log in to {CENTRAL} with the `admin` role.
