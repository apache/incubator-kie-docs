= Types of Columns in Guided Decision Tables

The *Add a new column* wizard for guided decision tables provides the following column options. (Select *Include advanced options* to view all options.)

* <<_guided_decision_tables_columns_condition_BRL_con, Add a Condition BRL fragment>>
* <<_guided_decision_tables_columns_attribute_con, Add a new Attribute column>>
* <<_guided_decision_tables_columns_metadata_con, Add a new Metadata column>>
* <<_guided_decision_tables_columns_condition_con, Add a simple Condition>>
* <<_guided_decision_tables_columns_action_BRL_con, Add an Action BRL fragment>>
* <<_guided_decision_tables_columns_delete_fact_con, Delete an existing fact>>
* <<_guided_decision_tables_columns_work_item_con, Execute a Work Item>>
* <<_guided_decision_tables_columns_field_con, Set the value of a field>>
* <<_guided_decision_tables_columns_field_work_item_con, Set the value of a field with a Work Item parameter>>

These column types and the parameters required for each in the *Add a new column* wizard are described in the sections that follow.

[id='_required_data_objects']
.IMPORTANT: Required data objects for column parameters
[IMPORTANT]
====
Some of the column parameters described in this section, such as Fact Patterns and Fields, provide drop-down options consisting only of data objects already defined within the same package where the guided decision table is found. Available data objects for the package are listed in the *Data Objects* panel of the Project Explorer and in the *Data Objects* tab of the guided decision table editor. You can create additional data objects within the package as needed, or import them from another package in *Data Objects* -> *New item* of the guided decision table editor. For details about creating data objects, see **(expand content from data-objects-create-proc.adoc).
