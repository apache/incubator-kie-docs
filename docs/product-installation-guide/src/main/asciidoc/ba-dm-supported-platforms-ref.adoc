= Supported platforms

You should install {CENTRAL} and {KIE_SERVER} on different servers in production environments.

The instructions in this document explain how to install {PRODUCT} on {EAP} 7.1 and Red Hat JBoss Web Server 5.0 on premise. For information about installing on {OPENSHIFT}, see
ifdef::DM[]
