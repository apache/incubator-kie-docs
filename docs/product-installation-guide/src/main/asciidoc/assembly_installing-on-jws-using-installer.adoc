= Using the installer to install {PRODUCT} on {EAP} or Red Hat JBoss Web Server 

This section describes the steps required to install {PRODUCT} using the installer JAR file. The JAR file is an executable file that installs {PRODUCT} in an existing {EAP} 7.1 or Red Hat JBoss Web Server 5.0 server installation. You can run the installer in standard or command line interface (CLI) mode.

[NOTE]
====
Because IBM JDK cannot use keystores generated on other JDKs, you cannot to install {PRODUCT} into an existing {EAP} installation running on IBM JDK with a keystore generated on another JDK.
====
