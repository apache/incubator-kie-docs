= Installing {PRODUCT} on {EAP} using the deployable ZIP files

Unlike Red Hat JBoss BRMS, the installation of {PRODUCT_VERSION} {RELEASE} on {EAP} consists of multiple deployable ZIP files, one for each component of the product, to provide additional flexibility.

For information on how to install {EAP} 7.1 on your computer, see the https://access.redhat.com/documentation/en-us/red_hat_jboss_enterprise_application_platform/7.0/html-single/installation_guide/[{EAP} Installation Guide].
