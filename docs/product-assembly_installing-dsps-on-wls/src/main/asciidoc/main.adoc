// Purpose statement for the assembly
As a system administrator, you configure your {WEBLOGIC} for Red Hat {KIE_SERVER} and install {KIE_SERVER} on that Oracle server instance.

.Prerequisites
* An {WEBLOGIC} instance is installed (release 12.2.1.3.0 or later). For installation instructions, visit the http://www.oracle.com/technetwork/middleware/weblogic/documentation/index.html[{WEBLOGIC} documentation page].
* You have access to the {WEBLOGIC} administrative console, usually `http://_TARGET_SERVER_:7001/console`.
