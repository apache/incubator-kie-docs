= Solving planning problems using the Business Optimizer

:imagesdir: topics/product-shared-docs/images
:context: business-optimizer

// Assembly purpose statement
As a business rules developer, you can use the Business Optimizer to find the optimal solution to planning problems based on a set of limited resources and under specific constraints.

.Prerequisite
The *Employee Rostering* sample project has been created in {CENTRAL} to demonstrate Business Optimizer capabilities.
