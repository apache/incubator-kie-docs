// Purpose statement
As a business rules developer, you can use the {PLANNER} to find the optimal solution to planning problems based on a set of limited resources and under specific constraints.

Use this document to set up and configure {PLANNER} functionality.

.Prerequisite
The *Employee Rostering* sample project is created in {CENTRAL} to demonstrate Business Optimizer capabilities. To view this project, you must have {PRODUCT} installed. For more information about setting up and installing {PRODUCT} and {CENTRAL}, see {URL_GETTING_STARTED_DECISION_SERVICE}[_{GETTING_STARTED_DECISION_SERVICE}_].
