= Intelligent Process Server Router Integration With Business Central

The Intelligent Process Server router provides a unified view of all servers. When integrated with Business Central, the management console allows you to do the following:

* Find the right server to manage requests.
* Aggregate responses from different servers.
* Provide efficient load balancing.
* Deal with a changing environment, such as servers being added or removed.
