// Purpose statement for the assembly
Decision Modeling Notation (DMN) is an open standard created by the Object Management Group (OMG). With DMN, organizations model their decision processes using a graphical language to build a diagram of decisions and requirements called a Decision Requirements Graph. This graph traces business decisions from start to finish, with each decision point drawing logic from custom tables known as decision tables.
