// Purpose statement for the assembly
As a business analyst or business rules developer, you can upload decision table spreadsheets to define business rules in a tabular format. These rules are compiled into DRL (Drools Rule Language) and form the core of the decision service for your project.

.Prerequisite
The team and project for the decision tables have been created in {CENTRAL}. Each asset is associated with a project assigned to a team. For details, see Getting started with decision services.
