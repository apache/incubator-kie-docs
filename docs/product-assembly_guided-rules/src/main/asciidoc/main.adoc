// Purpose statement for the assembly
As a business rules developer, you can define business rules using the guided rule designer in {CENTRAL}. These guided rules are compiled into DRL (Drools Rule Language) and form the core of the decision service for your project.

.Prerequisite
The team and project for the guided rules have been created in {CENTRAL}. Each asset is associated with a project assigned to a team. For details, see Getting started with decision services.
