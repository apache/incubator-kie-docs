= Dashboard data management in {CENTRAL}

ifdef::PAM,JBPM[]

{CENTRAL} contains dashboard and reporting tools that you can access from the data sets editor and the page editor. You can use these tools to work with the following types of items:

* Data sets
* Pages
* Navigation

You can export, import, and deploy dashboard data as ZIP files to and from {CENTRAL}.

IMPORTANT: This feature is only accessible by administrator users.

endif::[]
