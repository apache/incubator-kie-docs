== Embedded {CONTROLLER} calls

When running {CENTRAL} with the embedded {CONTROLLER} mode, a series of endpoints related to managing all aspects of {KIE_SERVER} templates, instances, and containers are also available. For more details, see <<controller-rest-api-con_kie-apis, {CONTROLLER} REST API>>.
A <<controller-java-api-con_kie-apis, Java client API>> is also available for interacting with these endpoints.
