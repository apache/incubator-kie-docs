== {PRODUCT} runtimes

=== {PRODUCT} Data Index service extends the Gateway API

A new set of GraphQL mutations are supported to perform the following operations on different {PRODUCT} services:

- Update process instance variables
- Trigger node instance related with a process instance
- Retrigger an existing node instance
- Cancel a node instance
- Reschedule job
- Cancel job

For more information, see {URL_CONFIGURING_KOGITO}#proc-data-index-service-runtime-connection_kogito-configuring[_{PRODUCT} Data Index Service Gateway API_].
